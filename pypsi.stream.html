7. pypsi.stream - File streams¶

Stream classes for writing to files.

class pypsi.stream.PypsiStream(fobj, prefix=None, postfix=None)[source]¶

Bases: object

Unbuffered file output stream. This class should be used primarily for writing to sys.stdout or sys.stderr. The stream allows for prefix and postfix strings to be written to the file before or after each call to write(). This can be useful when printing colors to the screen or the log levels of messages.

The wrapped file object can be a callable object that returns a file object. This object will then be called once during each invocation of write() to retrieve the current output file object.

Parameters:
  • fobj (file) – the file object to wrap or a function to retrieve the file object when writing to it
  • prefix (str) – a string to write to the stream whenever write() is called
  • postfix (str) – a string to write to the stream after write() is called
